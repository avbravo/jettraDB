# Jettra Web Management

A web-based management console for JettraDB cluster monitoring and data exploration.

## Features
*   **Dashboard**: Real-time stats of the cluster.
*   **Data Explorer**: UI to query and edit documents, graph vertices, and vectors.
*   **Cluster Management**: Monitor node health and replication status.
*   **REST API**: Exposes the database functionality via HTTP/JSON.

## Build
```bash
mvn quarkus:dev
```
Access at `http://localhost:8080`
