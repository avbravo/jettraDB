# Jettra Engine

The core storage engine of JettraDB. It provides the low-level API for all supported data models.

## Features
*   **Document Store**: JSON-based document storage with indexing.
*   **Vector Engine**: High-performance similarity search for AI/ML embeddings.
*   **Graph Engine**: Native vertex and edge storage for complex relationship traversal.
*   **KV Store**: Simple and fast key-value storage.

## Modules
- `jettra-engine-document`
- `jettra-engine-graph`
- `jettra-engine-vector`
